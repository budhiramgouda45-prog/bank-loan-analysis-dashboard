# Bank Loan Analysis Dashboard (Excel)

## Project Overview

I developed an interactive Bank Loan Analysis Dashboard using Microsoft Excel to analyze loan performance and provide actionable insights for better business decision-making.

---

## Business Problem

The bank didnt have any clear view of:

* Total loan applications
* Total funded amount
* Total amount received
* Loan quality (Good vs Bad loans)

This made it difficult for stakeholders to monitor performance and take data-driven decisions.

---

##Solution

To solve this problem, I built a dynamic and interactive dashboard in Excel that tracks key loan metrics, identifies trends, and highlights risk areas.I used Pivot Tables, slicers, and Excel charts to make the dashboard interactive and user-friendly.

---

## Dashboard Features

* KPI Cards → Total Applications, Funded Amount, Received Amount
* Donut Charts → Good Loans vs Bad Loans
* Bar Charts → Loan status, employment length, loan purpose
* Line Chart → Monthly loan application trends
* Map Visualization → State-wise loan distribution
* Treemap → Loan distribution by home ownership

---

##  Key Insights

* **Total Applications:** 38.6K
* **Total Funded Amount:** $435.76M
* **Total Received Amount:** $473.07M
* **Average Interest Rate:** 12.05%

### Loan Quality:

*  Good Loans: **86.18% (33.2K applications)**
*  Bad Loans: **13.82% (5.3K applications)**

### Key Observations:

* Majority loans (~32.1K) are fully paid → strong repayment behavior
* Loan applications show a steady increasing trend → business growth
* Top purposes:

  * Debt consolidation
  * Credit card
  * Home improvement
* Customers with **10+ years employment (~8.9K)** take more loans
* Customers in **rented homes (~18.43K)** take the most loans

---

##Business Recommendations

* Focus on reducing bad loans
* Strengthen loan approval checks:
  * Verify income sources
  * Review salary slips
  * Check credit score history

---

##Conclusion

This dashboard helps the bank to know the below informaton.
* Monitor loan performance
* Identify growth trends
* Reduce financial risk
* Make data-driven decisions
